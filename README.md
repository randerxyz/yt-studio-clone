# yt-studio-clone

Uma recriação fiel da interface do YouTube Studio feita com HTML, CSS e JavaScript puro — sem frameworks, sem dependências além do Chart.js.

Preview
<img width="1919" height="864" alt="image" src="https://github.com/user-attachments/assets/47d5b469-47c2-4ab1-8d2d-bcd5ab26738f" />

Objetivo

Projeto desenvolvido como estudo de HTML/CSS, portfólio, template reutilizável e demonstração de UI/UX. O objetivo foi replicar o visual e o comportamento do YouTube Studio o mais fielmente possível, incluindo animações, gráficos e interações.

Funcionalidades

- Navbar com logo, barra de busca centralizada e ações à direita
- Sidebar com avatar do canal, itens de navegação e scroll customizado
- Cards de estatísticas (visualizações, tempo de exibição, inscritos, receita)
- Gráfico de linha interativo com tooltip customizado e linha de hover
- Gráfico de barras em tempo real
- Menu de filtro de datas
- Animação de splash screen ao carregar
- Dot animado "Em tempo real"
- Visual dark fiel ao YouTube Studio

Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilização, animações, layout |
| JavaScript (vanilla) | Interatividade e lógica |
| [Chart.js 4.4.1](https://www.chartjs.org/) | Gráficos de linha e barras |
| Google Fonts (Roboto) | Tipografia |

Nenhum framework CSS ou JS foi utilizado. Tudo feito na mão. 💪
